  1. The population variance of a data set is a measure of “spread” denoted by

    \begin{equation*} V_P:=\frac{\sum (x_i-\mu)^2}{n} \end{equation*}

  2. A theoretically useful measure of spread with the same units as the original variable would be the population standard deviation denoted by

    \begin{equation*} \sigma:=\sqrt{V_P}=\sqrt{\frac{\sum (x_i-\mu)^2}{n}} \end{equation*}
