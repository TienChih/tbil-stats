Following the same reasoning as in SectionÂ 4.2 we can compute a confidence interval for the differences of true proportions. Suppose you had two categorical variables from whom you collect samples of size \(n_1, n_2\) and obtain sample proportions \(\hat{p}_1, \hat{p}_2\text{.}\) Then a C% confidence interval for the difference between the true proportions \(D=p_1-p_2\) is an interval centered at \(d=\hat{p}_1-\hat{p}_2\) which has a C% chance of containing D.
