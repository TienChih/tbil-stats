Suppose that we did have a random variable with \(k\) possible outcomes: \(O_1, \ldots O_k\text{,}\) and a sample with \(n_1, n_2, \ldots, n_k\) data points corresponding to each outcome. Note that the total size of the sample is \(n=n_1+\cdots n_k\text{.}\) Let \(E_1, \ldots, E_k\) denote the expected number of outcomes for each value, if a sample of size \(n\) were taken from the proposed distribution.

\begin{equation*} \begin{array}{|r|c|c|c|c|} \hline \text{Outcomes:} \amp O_1 \amp O_2 \amp \cdots \amp O_k \\ \hline \text{Sample Frequency:} \amp n_1 \amp n_2 \amp \cdots \amp n_k \\ \hline \text{Expected Frequency:} \amp E_1 \amp E_2 \amp \cdots \amp E_k \\ \hline \end{array} \end{equation*}

We then compute the test statistic for the \(i\)th value is

\begin{equation*} Z_i=\frac{n_i-E_i}{\sqrt{n_i}}. \end{equation*}

This essentially measures how much above or below the sample value is compared to the expected value, adjusted for sample size.
